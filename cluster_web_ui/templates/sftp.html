How to easily transfer files from/to Scale-Out Computing on AWS

Step1: Download FileZilla (or any other FTP app)

Mac Windows Linux
https://filezilla-project.org/download.php?platform=osx https://filezilla-project.org/download.php?platform=win64 https://filezilla-project.org/download.php?platform=linux

Step2: Configure FileZilla

Open FileZilla and select File > Site Manager to create a new Site
Host {{scheduler_ip}}
Port 22
Protocol SFTP
Logon Type Key File
User {{user}}
Key File /path/to/{{user}}_soca_privatekey.pem (click here to download your Linux private key if needed)
Save the settings and click connect

Step3: Connect and transfer file to FileZilla

During your first connection, you will be asked whether or not you want to trust {{scheduler_ip}}. Check "Always Trust this Host" and Click "Ok".
Once connected, simply drag & drop to upload/download files.
